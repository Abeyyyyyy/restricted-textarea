# Restricted Textarea

Project ini adalah latihan **DOM Manipulation menggunakan JavaScript** dari roadmap.sh.

Project URL:
https://roadmap.sh/projects/restricted-textarea

## Fitur

* Textarea untuk menulis pesan
* Live character counter
* Maksimum 250 karakter
* Counter bertambah secara real-time saat user mengetik
* Border textarea berubah merah ketika limit tercapai

## Teknologi

* HTML
* CSS
* JavaScript (DOM Manipulation)

## Cara Menjalankan Project

1. Clone repository ini

```
git clone https://github.com/Abeyyyyyy/restricted-textarea.git
```

2. Masuk ke folder project

```
cd restricted-textarea
```

3. Buka file `index.html` di browser.
